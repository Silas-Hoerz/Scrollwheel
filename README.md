# Scrollwheel (CY8CMBR3106S)

A robust and easy-to-use Arduino/PlatformIO library for a custom capacitive radial slider PCB based on the **CY8CMBR3106S** touch controller.

## Features
* **5-Segment Radial Slider:** Reads rotational data smoothly.
* **Central Touch Button:** Integrated central pad for selection/click actions.
* **I2C Interface:** Simple communication using standard standard I2C (SDA/SCL).
* **Advanced Kinematics (Example):** Includes a demonstration of advanced gesture extraction.
